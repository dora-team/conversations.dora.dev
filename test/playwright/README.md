# Playwright Tests

This directory contains end-to-end tests for the application using Playwright.

## Directory Structure

*   `playwright.config.ts`: Playwright configuration file.
*   `tests/`: Directory containing test files.

## Running Tests

To run the tests, navigate to this directory (`svelte/test/playwright`) and run:

```bash
npx playwright test
```

Or from the `svelte` root directory:

```bash
npx playwright test
```

## Writing Tests

Add new test files to the `tests/` directory.
